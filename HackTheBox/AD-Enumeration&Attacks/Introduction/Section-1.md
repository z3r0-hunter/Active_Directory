# What is Active Directory (AD)?

Directory Service used in Windows Enterprise environments
Provides centralized management of organizational resources and policies Resources it Manages:
 - Users
 - Computers 
 - Groups 
 - Network Devices 
 - File Shares 
 - Group Policies 
 - Trusts 

### AD Provide Three Core Functions:
- **Authentication** - Verifies user identity
- **Accounting**  - Logs and tracks activities
- **Authorization**  - Grants permissions and access rights

### AD Core Protocols:

- **X.500**: Provides hierarchical structure and tree-based architecture
- **LDAP**: Enables searching, communication, and data access

```md
AD = X.500 (Distributed & Hierarchical Structure) + LDAP (Search & Communication)
```
- History:
  - Officially released in 2000 with Windows Server 2000
  - Continuously improved with each new Windows Server version
